区域划分

课堂大概分为如下几个区域，内容水平居中

布局控制

通过在body上挂载class，来实现不同宽度或者边栏控制

  • body.l-min表示小宽度960px布局
  • body.l-max表示大宽度1200px布局
  • body.l-min.l-aside-left表示小宽度有左边栏布局，应用在个人中心管理
  • body.l-max.l-aside-left表示大宽度有左边栏布局，应用在机构管理
  • body.l-max.l-aside-right表示大宽度右边栏布局，应用在课程详细页

.inner-center

.inner-center应用于内容水平居中，如果body的class为l-min，则为960px水平居中；如为l-max，则为1200px水平居中

body.l-min(l-max,l-aside-left,l-aside-right)
.header>.inner-center
.main>.inner-center
.footer>.inner-center

全屏类型

全屏类型采用.section-name>.inner-center布局，如：

.section-feature>.inner-center // 应用于课程详细页和任务详细页的header之下的全屏内容
.section-postscript>.inner-center // 应用于课程详细页的课程推荐等footer之上的全屏内容

section

直接指定特殊化，适用单一情况

.section-name>.inner-center
    .section-tt
    .section-bd

多个section并列，可以先定义一个公用的class，再追加一个特殊化class

.section-area.area--name>.inner-center
    .section-tt
    .section-bd

如果标题右侧有内容，可按照上面的方式扩展一层section-hd

.section-name>.inner-center
    .section-hd // if 标题右侧有内容，嵌套一层hd
        .section-tt
        .hd-right // if 标题右侧内容
    .section-bd

main

采用负margin布局，content-wrap为100%的宽度

.main>.inner-center.clearfix
    .aside-left // if aside left
    .content-wrap>.content
    .aside-right // if aside right

aside

.aside-block.block--name
    .block-tt
    .block-bd

// 或

.aside-block.block--name
    .block-hd // if 标题右侧有内容，嵌套一层hd
        .block-tt 
        .hd-right // if 标题右侧内容

content

content中如有明显的标题引导内容，请使用以下结构，如课程详细页的课程描述

简洁版本

(.guide-tt+.guide-bd)*n

嵌套版本

.content-guide.guide--name
    .guide-tt
    .guide-bd

// 或

.content-guide.guide--name
    .guide-hd
        .guide-tt
        .hd-right
    .guide-bd
